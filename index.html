OPINX Predict 积分批量查询

批量查询指定钱包的指定周数，统计交易量、手续费成本、交易次数和积分。总交易量 = paid_volume_usdt + free_volume_usdt；成本 = paid_fee_usdt。
查询配置
支持自动去重。格式必须是 0x + 40 位十六进制字符。
支持：18、17,18、1-18、1-3,17,18
建议 3-10，太高可能被限流。
必须包含 {wallet} 占位符。
留空为直连。填写后会请求：代理前缀 + encodeURIComponent(真实 API URL)。
如果打开页面后请求失败并提示 Failed to fetch 或 CORS，说明接口没有允许浏览器跨域读取。单 HTML 页面无法绕过浏览器限制，需要你用自己的后端/Nginx/Cloudflare Worker 做一个转发代理。
